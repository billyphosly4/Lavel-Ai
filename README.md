# Lavel Ai

Lavel Ai is an intelligent, high-performance chatbot application built on Next.js.

## Features
- **Modern App Router**: Built entirely with React Server Components, Server Actions, and Next.js App Router.
- **Unified Intelligence**: Powered by seamless AI integrations.
- **Persistent Data**: Firebase Firestore backing chat history and memory.
- **Secure Access**: Fully fledged Firebase Authentication.

## Running locally

Copy the environment variables and ensure they are populated:
```bash
cp .env.example .env.local
```

Install packages and run the dev server:
```bash
pnpm install
pnpm dev
```
Your app should now be running on [localhost:3000](http://localhost:3000).
